Caddy with CloudFlare DNS module

## Installation

Download the binary for your architecture and install to `/usr/local/bin/caddy`:

### Linux AMD64 (x86_64)

```bash
curl -L -o /usr/local/bin/caddy \
  "https://github.com/lcmen/caddy-cloudflare/releases/download/latest/caddy-cloudflare-linux-amd64"
chmod +x /usr/local/bin/caddy
```

### Linux ARM64 (aarch64)

```bash
curl -L -o /usr/local/bin/caddy \
  "https://github.com/lcmen/caddy-cloudflare/releases/download/latest/caddy-cloudflare-linux-arm64"
chmod +x /usr/local/bin/caddy
```
